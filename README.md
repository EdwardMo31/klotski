# Klotski 华容道

一个纯前端、单文件的**华容道**（英文常称 *Klotski*）滑块拼图游戏。双击 `index.html` 即可离线游玩，无需任何依赖与构建。

> 在线试玩：https://EdwardMo31.github.io/klotski/

## 关于华容道

华容道是一款经典的滑块拼图游戏，主题取自《三国演义》"关羽义释曹操"——赤壁之战后曹操败走华容道。棋盘为 **4 列 × 5 行**（20 格），上面摆着不同大小的方块：

- **曹操**：2×2 大方块（目标是把他移出去）
- **关羽**：1×2 横块
- **张飞 / 赵云 / 马超 / 黄忠**：四个 2×1 竖块（五虎将）
- **卒**：四个 1×1 小兵

棋盘底部中间留有 2 格宽的出口。方块只能在空格内滑动，不能跨越或抬起。通过不断挪动其它棋子腾出空间，最终让曹操滑到底部出口即获胜。经典开局"横刀立马"看似简单，实则非常烧脑，是很好的逻辑与空间推理训练。

## 功能

- **19 个关卡，由易到难**：★ 入门（最优 1~17 步，熟悉手感）→ ★★ 进阶（40 余步，初尝迂回）→ ★★★ 大师（64~90 步，含横刀立马等经典开局）
- **两种操作方式**：点击棋子选中后用方向键，或直接拖动滑入空格
- **自动求解演示**：内置 BFS 求解器实时计算当前局面的最少步数并动画演示
- **提示**：高亮下一步该移动哪个棋子、往哪个方向
- **撤销 / 重开**，步数与计时
- **最佳成绩**：每关最少步数与最短用时本地保存（localStorage）
- **深 / 浅色主题**切换
- **移动端友好**：棋盘占据主屏，其余选项收纳进右上角 ⚙ 设置抽屉

## 计步说明

不同资料对华容道"步数"的计法不同：

- 每移动**一格**算一步：横刀立马最优为 116 步
- 每次**连续直线滑动**（同一棋子、同一方向滑到位）算一步：本程序采用此计法，由 BFS 实时求得各布局的最优步数
- 民间流传的"81 步"则采用允许棋子在一次移动中拐弯穿行空格的计法

本程序在界面上展示的"本关最优"即按"连续直线滑动 = 一步"的标准实时计算，与玩家计步规则一致。

## 技术

- 纯 HTML / CSS / JavaScript，**单文件**，零外部依赖
- 求解器在规范化网格字符串上做广度优先搜索（BFS），同型棋子等价合并以压缩状态空间
- 设置、最佳成绩持久化于 `localStorage`
